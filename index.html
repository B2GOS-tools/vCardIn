vCard Importer

  • This is a basic tool for importing all your contacts from a .vcf file to your phone. Due to FirefoxOS not allowing you to pick any file from your SDCARD to import, the way that this tool works is this:
  • 1. Name your .vcf file as contacts.vcf
  • 2. Drag and drop the file onto your SDCARD, putting it in the root path.
  • 3. Unplug your phone from the computer (this is important).
  • 4. Press the button Import contacts (see below).
  • Enjoy!
